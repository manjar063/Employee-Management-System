# Employee Management System

## Project Overview

Employee Management System is a console-based Java application developed using JDBC and Oracle Database. The project is designed to perform employee record management operations efficiently by connecting a Java application with a relational database. It allows users to add, view, search, update, and delete employee information through a simple command-line interface.

## Features

* Add new employee records to the database.
* Retrieve and display all employee details.
* Search for a specific employee using Employee ID.
* Update employee salary information.
* Delete employee records from the database.
* Handle exceptions such as duplicate Employee IDs and database connection errors.

## Technologies Used

* Java
* JDBC (Java Database Connectivity)
* Oracle Database
* SQL
* Eclipse IDE

## Key Concepts Implemented

* Database Connectivity using JDBC DriverManager
* CRUD Operations (Create, Read, Update, Delete)
* SQL Query Execution
* Exception Handling
* Object-Oriented Programming (OOP)

## Project Structure

The application establishes a connection with the Oracle Database and performs various employee management operations through SQL queries. User inputs are accepted using the console, and the corresponding database actions are executed using JDBC APIs.
